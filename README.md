# Archana Gundla | Software Developer Portfolio

A personal portfolio website showcasing my skills, experience, and projects as a software developer. Built using **HTML, Tailwind CSS, JavaScript**, and integrated with **EmailJS** for contact form functionality.

## Features

- Clean, responsive design with Tailwind CSS
- Sections: About, Skills, Experience, Projects, Contact
- Downloadable CV and live project demos
- Professional experience and education highlights
- Contact form using EmailJS
